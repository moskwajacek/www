O mnie

Niezwykle mi miło, że chcesz mnie trochę poznać - mam na imię Jacek i programowanie to moja największa pasja, ale zacznijmy od początku.
Od zawsze interesowały mnie przedmioty ścisłe, w szczególności królowa nauk matematyka. Poza tym zawsze fascynowała mnie informatyka. W liceum rozpoczęła się moja przygoda z programowaniem i tworzeniem stron www. Niestety życie ukierunkowało moją ścieżkę zawodową inaczej i przez wiele lat zajmowałem się czymś kompletnie innym. Przez długi okres czasu wahałem się z decyzją o przebranżowieniu. Dlaczego zdecydowałem się na ten krok? Lubię wyzwania.
Początkowo moją wiedzę czerpałem głównie z kursów internetowych oraz książek branżowych. Jednak czułem niedosyt i zdecydowałem się na intensywny kurs Front-End Developer. Wciąż uzupełniam moją wiedzę i szlifuje umiejętności programistyczne. Dążę do poszerzania swoich kompetencji poprzez poznawanie najnowszych rozwiązań i technologii programistycznych, a także doskonalenie nabytej już wiedzy. Dużą frajdę sprawia mi uczestnictwo w warsztatach branżowych. Posługuję się językiem angielskim na przyzwoitym poziomie. Dzięki czemu mogę szukać rozwiązań problemów na obcojęzycznych stronach internetowych. W przyszłości chciałbym zostać Full Stack Developerem.

Więcej na temat poznanych technologii opiszę w dziale poniżej.

Technologie

W tym dziale przedstawiam technologie, które poznałem. Jestem aktualnie na etapie poszerzenia wiedzy w zakresie biblioteki React technologii JavaScript. Chciałbym również zgłębić temat języka skryptowego PHP.

HTML 5

HTML oznacza Hypertext Markup Language - czyli hipertekstowy język znaczników.

CSS 3

Kaskadowe arkusze stylów to język służący do opisu formy prezentacji (wyświetlania) stron WWW.

JavaScript

JavaScript skryptowy język programowania, stworzony przez firmę Netscape.

JQuery

Lekka biblioteka programistyczna ułatwiająca korzystanie z języka JavaScript.

React

Biblioteka JavaScript do tworzenia interfejsów graficznych, aplikacji mobilnych i internetowych

Sass

Sass to preprocesor CSS, pozwala na znacznie szybszą i wydajniejszą pracę z arkuszami stylów.

Node JS

Środowisko uruchomieniowe do tworzenia wysoce skalowalnych aplikacji internetowych JavaScript.

Bootstrap

Framework CSS zawierający zestaw przydatnych narzędzi ułatwiających tworzenie interfejsu.

WordPress

System zarządzania treścią (CMS) przeznaczony głównie do obsługi blogów.

Git

Narzędzie, które służy do kontrolowanego wprowadzania zmian w projekcie.
